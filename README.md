# Vereinsraum – GitHub Pages

## Veröffentlichen

Diese Dateien gemeinsam in das Stammverzeichnis des GitHub-Repositories hochladen:

- `index.html`
- `manifest.webmanifest`
- `service-worker.js`

GitHub Pages anschließend auf den Branch `main` und den Ordner `/ (root)` einstellen.

## Installation auf iPhone/iPad

Web-App in Safari öffnen, **Teilen** wählen und **Zum Home-Bildschirm** antippen.

## Datenspeicherung

Auf GitHub Pages werden Daten lokal im jeweiligen Browser gespeichert. Unter **Backoffice → Backup** kann eine vollständige JSON-Sicherung heruntergeladen und wiederhergestellt werden. Gemeinsame geräteübergreifende Daten benötigen später einen kompatiblen `/api/storage`-Server; die App fällt ohne Server automatisch auf lokalen Speicher zurück.
